jd12_from_byear#

spacekernel.time.jd12_from_byear(ndarray byear) → ndarray#

Convert Besselian epoch back to two‐part Julian Date.

Parameters:
byearndarray of float, shape (N,)

Besselian epoch values.

Returns:
jd12ndarray of float, shape (N, 2)

Two‐part Julian Date [jd1, jd2] for each Besselian epoch, using iauEpb2jd from SOFA.
